Перевести вашу текущую реализацию repository слоя на использование spring data jpa.
То есть все ваши репозитории, которые обслуживают каждую из entity, должны быть
реализованы в рамках spring data jpa. Пагинационные методы получения сущностей,
должны быть реализованы с применением Pageable, а вся динамическая фильтрация
должна быть реализована через Specification
